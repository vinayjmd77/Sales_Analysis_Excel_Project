# Sales_Analysis_Excel_Project
Excel-based Sales Analysis Project featuring data cleaning, KPI generation, pivot table insights, and an interactive dashboard for business decision-making.

Overview:
This project focuses on analyzing sales data using Microsoft Excel. The goal was to transform a raw dataset into a clean, structured, and interactive dashboard that supports business decision-making.
The final output includes data cleaning, KPI calculations, pivot-based visualizations, and a professional dashboard for understanding sales trends and performance.

Project Files:

Dataset Description:
The dataset contains key sales-related attributes, including:
Customer Details: Customer ID, Gender, Age Group
Product Information: Category, Sub-Category
Transaction Details: Order Date, State, Quantity, Unit Price
Financial Metrics: Sales Amount, Profit
Total cleaned records after removing duplicates & errors: 4043 rows

Microsoft Excel
Data Cleaning (Filters, Sorting, Removing Duplicates)
Pivot Tables & Pivot Charts
Slicers & Timelines
Charts (Bar, Line, Pie, Map)

Data Cleaning Process:
Steps performed:
Removed duplicate and blank records
Standardized column formats (Text/Date/Numbers)
Corrected inconsistent category names
Filled missing values where applicable
Created new calculated fields for analysis
Verified final dataset (4043 rows)

Data Analysis & Visualizations:
Key analyses performed:
Total Sales & Profit Trend

Dashboard Overview:
The dashboard provides:
Total Sales
Total Profit
Customer Count
Total Quantity Sold

Monthly Sales Trend

It is fully interactive with slicers based on:
Year
State
Category

Insights & Findings:
Some important business insights derived:
Highest sales come from the East & South regions.
The Accessories category contributes major revenue.
Sales peak during festive months.
State-wise contribution shows imbalance—top 5 states drive most revenue.
Profit margin varies significantly by category.
These insights help in planning inventory, marketing, and target regions.
